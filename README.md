## 📖 Overview

NexusAI is an intelligent decision-support platform developed for the **Intelligent Next Best Action Platform Hackathon**.

The platform helps organizations analyze customer interactions, retrieve relevant business knowledge, and recommend the most effective next actions. Instead of functioning as a traditional chatbot, NexusAI leverages an agent-based architecture with contextual memory and explainable AI to assist business users in making informed decisions.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Saranyavellanki/NexusAI.git
```

Move into the project

```bash
cd NexusAI
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
